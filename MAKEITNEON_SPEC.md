# Make It Neon — Полная спецификация проекта

## Executive Summary

**Make It Neon** — информационный сайт-магазин компании по производству и продаже LED Neon Flex вывесок, базирующейся в Цюрихе, Швейцария. Сайт должен сочетать ультрасовременный дизайн уровня Awwwards с мощным функционалом: интерактивный калькулятор стоимости с визуализацией, каталог готовых вывесок, и систему заявок. Целевая аудитория — от владельцев малого бизнеса до event-агентств, по всей Швейцарии, Германии и миру.

## 2. Ценообразование и логика калькулятора

### 2.1 Как формируется цена неоновой вывески

Процесс производства определяет ценообразование:

```
Клиент вводит текст → Выбирает шрифт → Определяется размер →
→ Рассчитывается ДЛИНА ЦЕНТРАЛЬНОЙ ЛИНИИ букв (= длина LED-ленты = длина паза для фрезеровки) →
→ Учитывается длина контура (для вырезки подложки) →
→ Добавляются опции (цвет, RGB, outdoor, диммер) →
→ ИТОГОВАЯ ЦЕНА
```

### 2.2 Ключевые факторы цены

| Фактор | Важность (1-5) | Как влияет на цену |
|--------|:--------------:|-------------------|
| Длина LED-ленты (центральная линия букв) | ★★★★★ | ОСНОВНОЙ фактор. Больше ленты = больше материала + длиннее паз для фрезеровки |
| Размер вывески (ширина) | ★★★★★ | Определяет масштаб подложки из акрила |
| Кол-во символов | ★★★★★ | Прямо пропорционально длине ленты |
| Indoor / Outdoor (IP65+) | ★★★★★ | Outdoor требует герметизации, дороже материалы |
| RGB (цветосменяющий) | ★★★★★ | Дороже лента, контроллер, пайка |
| Логотип (ручная работа) | ★★★★★ | Индивидуальный расчёт, дизайнерская работа |
| Шрифт (сложность) | ★★★★☆ | Сложные шрифты = больше изгибов = длиннее лента |
| Диммер / пульт управления | ★★★★☆ | Фиксированная надбавка |
| Монтажные аксессуары | ★★★☆☆ | Фиксированные доплаты за тип крепления |
| Кол-во строк текста | ★★☆☆☆ | Влияет на высоту подложки |
| Тип подложки | ★★☆☆☆ | Только прозрачный акрил + опция аракал |
| Цвет LED (один цвет) | ★☆☆☆☆ | Одинаковая цена для всех одноцветных |

### 2.4 Техническая реализация калькулятора (детальный алгоритм)

Калькулятор работает на основе **измерения длины центральной линии (centerline/stroke path)** каждого символа текста, введённого пользователем. Это ключевой технический момент.

#### Алгоритм расчёта:

```
1. ВВОД: Пользователь вводит текст и выбирает шрифт
2. РЕНДЕР: Текст рендерится на Canvas (скрытый) с выбранным шрифтом
3. МАСШТАБИРОВАНИЕ: Текст масштабируется до выбранной ширины вывески
4. ИЗМЕРЕНИЕ CENTERLINE:
   - Для каждого символа в выбранном шрифте предварительно рассчитана
     длина центральной линии (stroke length) при базовом размере
   - При изменении масштаба — пропорциональный пересчёт
   - УПРОЩЁННЫЙ ПОДХОД: использовать таблицу "символ → множитель длины"
     для каждого шрифта, умноженный на высоту буквы
5. ЦЕНА ЛЕНТЫ: centerlineLength × pricePerCM_tape
6. ЦЕНА ФРЕЗЕРОВКИ: (centerlineLength + contourLength) × pricePerCM_milling
7. НАДБАВКИ:
   - RGB: +X% к базе
   - Outdoor (IP65): +Y% к базе
   - Диммер: +Z CHF (фиксированная)
   - Сложный шрифт: +W% к длине ленты
8. ИТОГ: basePriceFromTable + надбавки

АЛЬТЕРНАТИВНЫЙ ПОДХОД (РЕКОМЕНДУЕМЫЙ для MVP):
- Использовать Fixed Width Model (как у Sign Customiser):
  Фиксированные размеры вывески (40, 50, 60... 140 см)
  + максимальное кол-во символов для каждого размера
  + фиксированная цена из таблицы
  + надбавки за опции
```

#### Надбавки за опции (примерные, настраиваемые в admin):
| Опция | Тип надбавки | Примерное значение |
|-------|-------------|-------------------|
| RGB вместо одноцветного | % от базовой цены | +40-60% |
| Outdoor (IP65) | % от базовой цены | +30-50% |
| Диммер + пульт | Фиксированная | +25-35 CHF |
| Настенное крепление (стандарт) | Включено | 0 |
| Подвесное крепление | Фиксированная | +15-25 CHF |
| Стойка/подставка | Фиксированная | +35-55 CHF |
| Аракал (заливка цветом подложки) | Фиксированная | +20-40 CHF |
